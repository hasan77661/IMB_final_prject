## Third: Top 10 Common Data Issues & How to Fix Them in Excel for the Web

| #      |                   Issue                         | Practical Cleaning Steps in Excel for the Web                                                                                                                                                |
===========================================================================================================================================================================================================================================================
| ---    | ---                                             | ---                                                                                                                                                                                          |
| **1**  | **Missing Values (Blanks)**                     | Turn on Filters (`Ctrl + Shift + L`) ➔ Choose **(Blanks)** ➔ Highlight the blank rows ➔ Right-click and select **Delete Rows**.                                                             |

| **2**  | **Duplicate Rows**                              | Select the entire dataset (`Ctrl + A`) ➔ Go to the **Data** tab ➔ Click **Remove Duplicates** ➔ Click **OK**.                                                                               |

| **3**  | **Hidden Spaces (Leading/Trailing)**            | Insert a temporary column ➔ Write `=TRIM(A2)` ➔ Drag it down, then copy the result and **Paste Values** over the original text.                                                             |

| **4**  | **Numbers Stored as Text**                      | Highlight the column ➔ Go to the **Home** tab ➔ Open the formatting dropdown (usually says *General*) ➔ Select **Number**.                                                                  |

| **5**  | **Mixed Date Formats**                          | Select the date column ➔ Go to the **Home** tab ➔ Open the formatting dropdown ➔ Choose **Short Date** to unify them instantly.                                                             |

| **6**  | **Outliers (Anomalous Figures)**                | Filter your Sales or Price columns ➔ **Sort Largest to Smallest** to quickly isolate and review massive, illogical numbers for correction or deletion.                                       |

| **7**  | **Inconsistent Casing**                         | Use `=UPPER(A2)` in a helper column to standardize promo or product IDs (e.g., converting `pr14` to `PR14`).                                                                                 |

| **8**  | **Garbled CSV Text Encoding**                   | If characters look like broken symbols (e.g., ``), re-upload the file to OneDrive ensuring the File Origin/Encoding is set to **UTF-8**.                                                     |

| **9**  | **Formula Errors (e.g., `#VALUE!`, `#DIV/0!`)** | Wrap your equations using `=IFERROR(Your_Formula, 0)` to cleanly catch math calculation issues and replace errors with a neutral zero or blank space.                                        |

| **10** | **Completely Empty Columns**                    | In promo tracking data (like `promo_bin_2`), if an entire column is completely empty across thousands of rows, right-click the column letter and select **Delete Column** to trim file size. |
